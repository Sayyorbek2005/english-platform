1. Front-end (Foydalanuvchi interfeysi) Bu qism foydalanuvchi ko‘radigan va o‘zaro ishlaydigan qismdir. Texnologiyalar: HTML – sahifaning tuzilmasini yaratadi. CSS – dizayn, rang, joylashuv va tartib uchun. JavaScript – interaktivlik qo‘shadi (masalan, tugma bosilganda biror narsa sodir bo‘lishi). Kutubxonalar va freymvorklar: React, Vue, Angular – murakkab ilovalar yaratishda yordam beradi.



2. Back-end (Server tomoni) Bu qism foydalanuvchi ko‘rmaydigan, ammo saytni ishlashiga sabab bo‘ladigan tizimdir. Dasturlash tillari: Node.js (JavaScript), Python (Django, Flask), PHP, Ruby, Java, C# va boshqalar. Ma’lumotlar bazasi: MySQL, PostgreSQL, MongoDB – ma’lumotlarni saqlash uchun. Server va API’lar: Serverda kod ishlaydi, foydalanuvchining so‘rovlariga javob beradi (masalan, login qilish).



3. Full-stack Developer Front-end ham, back-end ham bilan ishlay oladigan dasturchi.



4. Qo‘shimcha tushunchalar: Git va GitHub – kodni boshqarish va jamoa bilan ishlash uchun. Hosting va domen – saytni internetda joylashtirish. Responsiveness – sayt turli ekranlarda to‘g‘ri ko‘rinishini ta’minlash.
